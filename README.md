# Своя игра

Учебный веб-проект на Spring.

## Идея проекта
Приложение по мотивам игры «Своя игра», где можно:
- выбирать темы
- выбирать стоимость вопроса
- отвечать на вопросы
- считать очки

## Стек
- Java
- Spring Boot
- Maven
- Thymeleaf
- PostgreSQL
