# Своя игра

Учебный Spring Boot веб-проект в формате телевизионной игры «Своя игра».

## Что уже есть

- Spring Boot приложение с веб-слоем на `Spring MVC`
- серверный рендеринг через `Thymeleaf`
- игровое поле с категориями и вопросами
- in-memory сервис с демо-данными
- базовые доменные сущности без БД
- стартовые тесты на поднятие контекста

## Текущий стек

- Java 21
- Spring Boot 3.3
- Maven
- Thymeleaf
- CSS
